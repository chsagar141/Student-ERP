# 🎓 Student ERP Web Application

A simple and efficient Student ERP (Enterprise Resource Planning) system designed to manage student records, subjects, and marks. This project focuses on a clean backend architecture with a lightweight frontend.

🚀 Features
📌 Student management system
📚 Subject allocation
📝 Marks tracking
🔐 Secure authentication with password hashing
⚡ Fast and lightweight backend
🛠️ Tech Stack
Backend (Developed by Me)
FastAPI (high-performance Python framework)
SQL (Relational Database)
Password Hashing for secure authentication
Frontend
Built using HTML & CSS
Assisted heavily with AI-generated components
⚠️ Important Note

🔹 The backend is fully designed and implemented by me, including:

API development
Database schema design
Authentication & security (password hashing)

🔹 The frontend is mostly AI-assisted, focused on simplicity rather than complex UI/UX.
